# YouTube Scripts（動画スクリプト）

このディレクトリは、YouTube動画用の台本・スクリプト作成のために使用します。

---

## ディレクトリ構成

```
youtube-scripts/
├── README.md                          # 本ファイル
├── fx-automation-lab/                 # はかせのFX自動売買研究所
│   └── README.md                      # チャンネル詳細・カテゴリ解説
└── fx-discretionary-method/           # はかせのFX裁量手法チャンネル
    └── README.md                      # チャンネル詳細・カテゴリ解説
```

## チャンネル別の詳細

### 1. はかせのFX自動売買研究所（`fx-automation-lab/`）

**コンセプト**：EA検証・分析を通じた実践的情報の提供

**主なカテゴリ**：
- EA検証動画
- HACHI EA関連
- GOLD EA研究所（コミュニティ企画）
- 代理店EA紹介
- 自動売買の基礎知識
- アップデート・告知

詳細は `fx-automation-lab/README.md` を参照。

---

### 2. はかせのFX裁量手法チャンネル（`fx-discretionary-method/`）

**コンセプト**：裁量トレードの本質的な教育

**主なカテゴリ**：
- ダウ理論
- MTF（マルチタイムフレーム分析）
- トレンド分析
- サポート・レジスタンス
- エントリー・決済の技術
- リスク管理とメンタル
- 実践トレード解説
- チャートパターン
- インジケーター活用
- 通貨ペア別攻略

詳細は `fx-discretionary-method/README.md` を参照。

## チャンネル間の関係性

両チャンネルは相互補完的な関係：

| 項目 | 自動売買研究所 | 裁量手法チャンネル |
|------|----------------|-------------------|
| 焦点 | EA・自動売買 | 裁量トレード |
| 視聴者層 | 自動化志向 | スキル習得志向 |
| 収益モデル | 販売・IB（確立済） | 教材化検討中 |
| ブランド価値 | 実績・信頼性 | 教育力・専門性 |

→ 視聴者の幅を広げ、ブランド全体の価値を向上させる

## スクリプト作成の共通ルール

### EA検証スクリプト（自動売買研究所）

プロジェクトルートの `README.md` に記載されている  
**「EA検証スクリプト用ガイドライン」** を必ず参照してください。

**重要な思想**：
- データベース（定量）＋ストーリー（定性）のハイブリッド
- 共感 × 分析 × 中立性のバランス
- 誤解なく・過信させず・自己判断力を育てる

### 教育コンテンツ（裁量手法チャンネル）

- 本質を伝える（原理原則重視）
- 実践的に教える（チャート事例必須）
- 段階的に学べる（初心者→中級者）
- 考える力を育てる（答えを与えすぎない）

## ファイル命名規則

各チャンネルのディレクトリ内で、以下の形式を推奨：

```
YYYY-MM-DD_カテゴリ_タイトル.md
```

**自動売買研究所の例**：
- `2025-12-26_EA検証_タートルEA_12月成績.md`
- `2025-12-26_HACHI_最新アップデート情報.md`

**裁量手法チャンネルの例**：
- `2025-12-26_ダウ理論_トレンド転換の見極め方.md`
- `2025-12-26_MTF_環境認識の基本手順.md`

## Webアプリ事業との連携可能性

両チャンネルとも、分析ツール（Webアプリ事業）との連携が可能：

- **自動売買研究所**：HACHI向けモニタリングツール、EA分析ダッシュボード
- **裁量手法チャンネル**：ダウ理論可視化ツール、トレンド判定アプリ、相場分析ツール

動画内でツールを紹介し、実践的な使い方を解説することで、  
コンテンツの価値を高め、新たな収益源の構築も可能。
