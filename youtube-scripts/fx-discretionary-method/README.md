## チャンネルコンセプト

## 動画カテゴリ

## 教育スタイル

### 基本方針
- **本質を伝える**：小手先のテクニックではなく、原理原則を重視
- **実践的に教える**：理論だけでなく、実際のチャートで解説
- **段階的に学べる**：初心者から中級者まで段階的にステップアップ
- **誤解を生まない**：曖昧な表現を避け、明確に説明
- **考える力を育てる**：答えを与えすぎず、判断力を養う

### トーンガイド
- 教育的・丁寧・分かりやすく
- 専門用語は必ず解説
- 上から目線にならない
- 失敗例も隠さず共有
- 「聖杯はない」ことを伝える
