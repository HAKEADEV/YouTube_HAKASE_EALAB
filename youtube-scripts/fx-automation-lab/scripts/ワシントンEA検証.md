## ③ロジックの解説（性格の翻訳）

通貨：GOLD
時間足：M15
推奨ロット：3万円ごとに0.1 lot(マイクロ口座)
想定月利：60%

バックテストでは100万円 → 464万円 (+364万円)

とのこと
「これだけの月利で初心者にも安全なのか」
「リスクはないのか」
このあたりをバックテストで検証していきたいと思います。

---

## ④検証条件（バックテストの前提）

「今回は広告で書かれていた条件を参考に、以下の設定で検証しました。

通貨：GOLD（XAUUSD）

期間：2024年1月〜2025年9月末

ロット：0.1（マイクロ口座換算）

証拠金：まず3万円、次に15万円の2パターン

この条件で、"少額から安全にできる"という広告文の真偽を確かめます。」

---

## ⑤結果①：平均月利と含み損の傾向

「まず大前提として、これは完全なナンピンマーチンEAです。
単ポジではありませんし、安全型でもありません。

推奨証拠金3万円で運用した場合、
破綻回数が異常な数検出されました。

グラフだけを見ると右肩上がりなんですが、
これは"余剰資金が無限にある前提"でしか成り立ちません。
実際に3万円で回したら、ほぼ破綻祭りです。

なので、"少額からできる"という宣伝文句は、
裏を返せば"破綻者を続出させる構造"とも言えます。」

「次に、もう少し現実的な条件として、
証拠金15万円・マイクロ口座運用を想定して見てみましょう。

2025年9月8日には、含み損955ドル、
ほぼ飛ぶか飛ばないかのギリギリラインを記録しました。

このときの月利が**約43％**なので、
攻撃力自体は高く、利益率のポテンシャルは確かにあります。

ただし、2025年1月から10か月の間で4〜5回破綻しているため、
トータル期待値で見るとトントン〜微マイナスといった結果でした。」

「さらに"岩ライザー"で含み損の推移を分析したところ、
2024年以降、含み損を抱える頻度と金額が急増しています。
つまり、直近のGOLD相場に対応しきれていない。

ここ最近のボラティリティ変化に対して、
完全に最適化が遅れているEAと言えるでしょう。」

---

## ⑥チャートで確認（リスクの実感）

「実際のチャートで見てみましょう。

含み損を大きく抱えた代表的なシーンがこちら。
2025年9月8日付近では、
300〜400pipsの逆行で約955ドルの含み損を抱えています。

そのときポジション数は15前後。
1ポジ0.01lotであっても、合計ロットで0.15。
この状態になると、マーチンロジック特有の"延命戦法"が働いているのがわかります。

つまり、"ロスカットにならない"ではなく、
**"ロスカットを引き延ばしている"**だけの状態。
この差は非常に大きいです。」

---

## ⑦曜日別傾向の確認

「曜日別の損益を見ていくと、
水曜日と金曜日がまだマシな結果でした。

ただ、平均獲得pipsは全体的にマイナス。
これは、逆行したところをマーチンで取り返す構造だからですね。

つまり、水曜・金曜だけ回したとしても、
破綻する可能性は十分にあります。
部分的な稼働でリスクを軽減できるタイプではなさそうです。」

---

## ⑧このEAの本質（広告の裏側）

「ここまでの結果を振り返ると、
この"ワシントンEA"が伝えようとしているメッセージと、
実際の中身はかなり乖離しています。

| 広告コピー | 実際の挙動 |
|----------|-----------|
| ナンピンしない | 実際はしっかりナンピンマーチン構造 |
| ロスカットにならない | ロスカット"しにくいように延命"するだけ |
| 単ポジ型 | 最大15ポジ程度積み上げ |
| 少額からできる | 少額ほど即破綻 |
| 放置OK | 相場を見ずに放置すると確実に飛ぶ |

つまり、"初心者向け"と謳いつつ、
実際はかなり上級者向けのEAです。

短期的には右肩上がりのバックテストが出ますが、
それは"破綻前提の爆益構造"にすぎません。」

---

## ⑨クロージング＆まとめ

今回の結果をまとめると、こうなります。

| 項目 | 結果 |
|------|------|
| 構造 | ナンピンマーチンEA |
| 推奨証拠金3万円 | 破綻多数、検証不能 |
| 証拠金15万円 | 月利43％・含み損955ドル（ギリ耐え） |
| 破綻頻度 | 10か月で4〜5回 |
| 稼働安定曜日 | 水・金（それでも危険） |
| 期待値 | トントン〜微マイナス |
| 安全性評価 | ✕（高リスク型） |

結論、広告の"安全・放置・単ポジ"という表現は誤解を招くと思います。
実態は、"高攻撃力・高リスク"の典型的マーチンEA。

特に最近のGOLDのボラティリティでは、
破綻の頻度が急増している点には注意が必要です。

運用する場合は、
・最悪全損しても良い余剰資金で
・定期出金を前提に
・トレンド転換時は必ず停止

この3つを守らないと、破綻リスクが避けられません。

"放置で稼げるEA"という言葉に惹かれがちですが、
本当に放置できるEAなんて存在しません。
リスクを管理して初めて、EAは"使えるツール"になる。
