# Scripts（過去のスクリプト保存）

このディレクトリは、「はかせのFX自動売買研究所」で実際に使用したスクリプトを保存します。

---

## 保存する目的

1. **ナレッジの蓄積**：成功したスクリプトのパターンを分析
2. **テンプレート化**：効果的だった構成を再利用
3. **改善の参考**：過去の反省点を次回に活かす
4. **一貫性の維持**：トーンやスタイルの統一

---

## ファイル命名規則

```
YYYY-MM-DD_カテゴリ_タイトル_公開状況.md
```

**例**：
- `2025-12-26_EA検証_タートルEA_12月成績_公開済.md`
- `2025-12-26_HACHI_最新アップデート_下書き.md`
- `2025-12-25_GOLD研究所_フォワード大会結果_公開済.md`

---

## ディレクトリ構成（推奨）

カテゴリごとにサブディレクトリを作成することも可能：

```
scripts/
├── README.md
├── ea-verification/          # EA検証スクリプト
├── hachi/                    # HACHI関連
├── gold-lab/                 # GOLD EA研究所
├── agent-ea/                 # 代理店EA
└── education/                # 自動売買の基礎知識
```

必要に応じて自由に構造を変更してください。

## スクリプトに含めるべき情報

各スクリプトファイルには以下の情報を記載することを推奨：

### メタデータ（ファイル冒頭）
```markdown
---
公開日: 2025-12-26
動画URL: https://youtube.com/watch?v=xxxxx
カテゴリ: EA検証
視聴回数: 12,345回（2025-12-30時点）
高評価率: 95%
主なコメント: 「わかりやすい」「リスク解説が良い」
反省点: 冒頭が長かった、チャート表示をもっと大きく
次回への改善: オープニングを30秒以内に
---
```

### スクリプト本文
- 実際に使用したナレーション内容
- タイムスタンプ（各セクションの時間）
- 使用したチャート・資料の説明

---

## 振り返りのポイント

スクリプトを保存する際は、以下を意識して記録：

✅ **うまくいった点**
- どの構成が視聴者に刺さったか
- どの表現がわかりやすかったか

❌ **改善点**
- 冗長だった部分
- 説明不足だった箇所
- テンポが悪かった場所

💡 **次回への活かし方**
- テンプレート化できる構成
- 使い回せる表現・フレーズ

---
