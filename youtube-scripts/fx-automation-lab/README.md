# はかせのFX自動売買研究所

このディレクトリは「はかせのFX自動売買研究所」チャンネル用の動画スクリプトを管理します。

---

## チャンネルコンセプト

EA（自動売買）を中心とした、検証・分析・研究を通じて視聴者に実践的な情報を届けるチャンネル。

---

## 動画カテゴリ

### 1. EA検証動画
**目的**：EAのパフォーマンスを客観的に分析し、視聴者の判断材料を提供

- バックテスト結果の詳細解説
- フォワードテスト報告
- 月次・年次の成績レビュー
- リスク分析（最大含み損・連敗数）
- 曜日別・時間帯別の傾向分析
- EA比較・ポートフォリオ検討

**トーンガイド**：
- データベース（定量）× ストーリー（定性）のハイブリッド
- 過信させず、誤解なく、自己判断力を育てる
- 共感 × 分析 × 中立性のバランス

### 2. HACHI EA関連
**目的**：主力商品HACHIの価値提供とアップデート情報の発信

- HACHIのロジック解説
- 最新バージョンのアップデート内容
- 運用実績報告
- 日本口座版 vs 海外口座版の違い
- 月1キャンペーン告知
- ユーザー事例・Q&A

### 3. GOLD EA研究所（コミュニティ企画）
**目的**：視聴者参加型の研究・開発を通じたコミュニティ形成

- GOLD EA開発プロジェクトの進捗報告
- 破綻研究の結果共有
- フォワード大会の結果発表
- コミュニティ内の議論・気づきのシェア
- オープンチャットの活動報告

### 4. 代理店EA紹介・検証
**目的**：取り扱いEAの紹介と客観的評価

- タートルEA
- owl
- LISA
- その他取り扱いEAの検証・紹介

### 5. 自動売買の基礎知識
**目的**：初心者向けの教育コンテンツ

- EAの選び方
- リスク管理の基本
- 証拠金設計
- ロット計算
- バックテストの見方
- フォワードテストの重要性
- 相場環境とEAの相性

### 6. アップデート・告知
**目的**：視聴者への最新情報の提供

- 新規EA発表
- キャンペーン情報
- 規約変更・注意喚起
- 証券会社情報

### 7. マーケット分析（EA視点）
**目的**：相場環境とEAパフォーマンスの関連性を解説

- 今月の相場振り返り
- EA運用における注意点
- 相場環境の変化とEA調整
- イベントリスクの解説

## スクリプト作成の基準

プロジェクトルートの `README.md` に記載されている  
**「EA検証スクリプト用ガイドライン」** を必ず参照してください。

### 8つの構成セクション
1. オープニング（信頼性＋ギャップ提示）
2. なぜ検証したのか（動機と共感づくり）
3. ロジックの解説（EAの性格を翻訳）
4. バックテストの月別成績
5. リスクパート（含み損・最大連敗）
6. チャートでリスクの高い場面を確認
7. 曜日別傾向の確認
8. クロージング＆まとめ

## ファイル命名規則

```
YYYY-MM-DD_カテゴリ_タイトル.md
```

**例**：
- `2025-12-26_EA検証_タートルEA_12月成績.md`
- `2025-12-26_HACHI_最新アップデート情報.md`
- `2025-12-26_GOLD研究所_フォワード大会結果.md`
