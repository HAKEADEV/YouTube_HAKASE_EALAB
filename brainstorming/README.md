# Brainstorming（壁打ち）

このディレクトリは、事業全体の戦略相談・アイデア出し・ナレッジ蓄積のために使用します。

## 使い方

- 新規企画の相談
- 事業レイヤー間の関係性整理
- リスク分析・戦略立案
- EA検証方法の検討
- マーケティング施策の壁打ち
- 技術的な実装相談
- マインドマップ作成（思想の広がり）

## ファイル命名規則（推奨）

- `YYYY-MM-DD_テーマ.md`
- 例：`2025-12-26_HACHI海外版戦略.md`

## 構造（自由）

プロジェクトごと、テーマごとに自由にサブディレクトリを作成してください。
全てのプロジェクトをここで管理する必要はなく、必要に応じて別リポジトリへ分割も可能です。
